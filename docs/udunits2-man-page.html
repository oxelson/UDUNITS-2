    udunits2 Man Page

    UDUNITS-2 Documentation udunits2 Man Page

    This page provides the Markdown equivalent of the udunits2(1) man page, documenting command-line usage, options, supported expression syntax, and examples for the udunits2 CLI utility.

    It serves as a reference for users who prefer web-based or repository documentation formats while maintaining consistency with traditional Unix manual content.

    NAME

    udunits2 — convert values between compatible units and inspect unit definitions using the UDUNITS-2 library

    SYNOPSIS

    udunits2 [options]

    udunits2 -H "<from-expression>" -W "<to-units>"

    DESCRIPTION

    udunits2 is a command-line utility that uses the UDUNITS-2 library to:

    • convert numeric values between compatible units
    • determine dimensional compatibility between units
    • display unit definitions in terms of base units
    • interactively explore unit relationships

    The program reads unit definitions from the UDUNITS XML database and applies the same parsing and conversion logic used by the C API (ut_parse(), ut_get_converter(), etc.).

    By default, the program operates in interactive mode, prompting the user to enter unit expressions for conversion.

    OPTIONS

    Conversion options

    -H from-expression

    Specifies the input quantity, including both numeric value and units.

    Example:

    -H "10 m/s"

    -W to-units

    Specifies the units to convert to.

    Example:

    -W "km/h"

    Database options

    -L path

    Specifies the XML unit database file to load instead of the default.

    Example:

    -L /usr/local/share/udunits/udunits2.xml

    Formatting options

    -h

    Displays usage help.

    -V

    Displays program version information.

    INTERACTIVE MODE

    When invoked without conversion arguments, udunits2 enters interactive mode:

    udunits2

    Example session:

    You have: 1 meter
You want: feet
    1 meter = 3.28084 feet

    EXPRESSIONS

    Quantity expressions

    A quantity expression combines a numeric value with a unit:

    15 m/s
9.81 m/s^2
1e3 kg

    Unit expressions

    Unit expressions may include:

    • multiplication
    • division
    • exponents
    • prefixes
    • derived units

    Examples:

    kg*m/s^2
W/m^2
km/hour

    EXAMPLES

    Convert meters per second to kilometers per hour

    udunits2 -H "10 m/s" -W "km/h"

    Output:

    10 m/s = 36 km/h

    Convert degrees Celsius to Fahrenheit

    udunits2 -H "0 degC" -W "degF"

    Display dimensional definition of a unit

    udunits2 -H "W" -W ""

    Use a custom unit database

    udunits2 -L ./udunits2.xml

    EXIT STATUS

    Status Meaning
    0 Successful execution
    non-zero Error occurred (parse error, incompatible units, file error, etc.)

    FILES

    Typical unit database location:

    /usr/share/udunits/udunits2.xml
/usr/local/share/udunits/udunits2.xml

    ENVIRONMENT

    UDUNITS2_XML_PATH

    Specifies the search path for the XML unit database.

    If not set, the library searches default installation locations.

    NOTES

    • Unit definitions are based on the UDUNITS-2 XML database.
    • Conversion results depend on dimensional compatibility.
    • Logarithmic and offset units (e.g., °C, dB) are handled according to UDUNITS-2 semantics.
